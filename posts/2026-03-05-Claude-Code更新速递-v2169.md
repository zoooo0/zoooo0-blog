> **写在前面**：Claude Code 更新速递第八期。上期聊了 Opus Medium Effort 和 8 处内存泄漏修复，这期只覆盖一个版本号——v2.1.69，但这一个版本的更新日志比上期六个版本加起来还长。光内存泄漏又修了 13 个，安全漏洞堵了 4 个。Anthropic 这更新节奏，属实是把"版本号"当"专题期刊"在发。

---

## 内存泄漏大扫除（第二季）

上期修了 8 个，这期又来 13 个。不是产品质量差，而是 Anthropic 在系统性清技术债。

新特性越堆越多——Agent Teams、长会话、多 MCP 并行——底层的内存管理就被暴露得越彻底。越用越深，漏洞就越藏不住。

![内存泄漏](https://files.catbox.moe/r16yx8.jpg)

挑几个典型的：

React Compiler 的 memoCache 只增不减，长会话跑久了界面卡顿，根源在这儿。REPL 的渲染作用域也不回收，实测 1000 轮对话泄漏约 35MB，单看不多，但这只是十三个泄漏源之一。

Agent Teams 场景下有个更夸张的：每个子 Agent 都完整复制了父级的对话历史。你开三四个子 Agent 跑一阵，内存不飙才怪。改成只保留必要上下文后好多了。

还有 hook events 注册了不注销（经典老问题），以及提交大文件时内存瞬间飙到几个 GB（现在改成流式处理，峰值砍了一大截）。

堵完漏还做了一轮"减肥"：Yoga WASM 延迟加载，基线内存减少约 16MB；大会话恢复和 SDK 的内存分配也做了优化。

说实话，我这种一个会话开半天不关的人，升完级最大的感受就是不卡了。比新功能管用。

## 安全加固四连击

这波修了 4 个安全漏洞，都是实打实能被利用的。

![安全加固](https://files.catbox.moe/u22r17.jpg)

最值得关注的是嵌套 skill 的路径解析——之前没做限制，恶意 npm 包可以从 node_modules 注入 skill。现在严格限制了加载路径。另一个是 acceptEdits 模式下的符号链接逃逸，通过符号链接能写到项目目录外面的文件，开了自动接受编辑的团队要留意。

还有两个权限相关的：首次运行时 .mcp.json 里的服务器会被静默全部启用，现在改成逐个确认；skill 的 allowed-tools 里塞了交互类工具会绕过权限提示，也堵上了。

用了第三方 skill 或开了 acceptEdits 的团队，建议尽快升级。

## Sonnet 4.5 谢幕，4.6 接棒

Pro、Max、Team Premium 用户的 Sonnet 4.5 自动迁移到 Sonnet 4.6。不需要手动操作，/model 里会直接看到变化。

4.6 是当前最新的 Sonnet，编码性能更好，上下文处理更稳。如果你一直用 Sonnet 做日常开发，这次迁移是纯受益，不用纠结。

## 其他更新一览

![其他更新](https://files.catbox.moe/3yoysz.jpg)

**/claude-api 内置技能**。新增官方 skill，帮你用 Claude API 和 Anthropic SDK 写应用。以后要接 Claude API，直接跟 Claude Code 说就行，不用自己翻文档。

**Effort level 可视化**。logo 和 spinner 现在会显示当前的 effort level（比如 "with low effort"）。不用猜 Claude 是在认真想还是在敷衍了。

**Voice STT 新增 10 种语言**。新增俄、波兰、土耳其等 10 种语言，总计支持 20 种。

**${CLAUDE_SKILL_DIR} 变量**。Skill 里可以引用自身所在目录，不用再写绝对路径。

**InstructionsLoaded hook**。CLAUDE.md 或 rules 文件加载时触发，企业合规审计场景有用。

**includeGitInstructions 设置**。关闭内置 git 指令，给深度定制用户腾出系统提示词空间。

**/reload-plugins**。插件或 MCP 配置变更后不用重启 Claude Code，直接刷新生效。

**MCP 二进制内容处理优化**。PDF、Office 文档、音频等二进制内容现在保存到磁盘而不是 base64 灌进上下文。context window 终于不会被一个大文件撑爆了。

**性能优化若干**。React Compiler 渲染优化、spinner 动画隔离减少 CPU 开销、macOS 启动加速、MCP 连接流水线化。

**VSCode / Windows 修复**。VSCode 侧压缩聊天可折叠、权限选择器尊重策略设置；Windows 侧修复 worktree 文件复制、.claude 目录检测等。
